# Development

## Haskell(GHC) environment

Since Ganeti 3.0.0, the upstream release is built/tested against GHC 8.0 through 8.8 which means it should work on most current and near-future distribution versions. Support for GHC versions < 8 has already been dropped with the previous Ganeti release.

The GHC versions provided by the EPEL repositories for el7 and el8 are as follows:

- el7: 7.6.3-x.y.el7
- el8: 8.2.2-x.y.el8

GHC for el7 will not build Ganeti 3.0.0 or later. In addition, we have been built and provided RPM packges for many GHC libraries required to build Ganeti, but their version dependencies are complex and very difficult to build.

For those reasons, starting with Ganeti 3.0.0 RPM package, we decided to build using [the GHC's official Binary Packages](https://www.haskell.org/ghc/download.html).

## SELinux policy

Starting with `v3.1.1-2`, the RPM installs a Ganeti SELinux policy module.

After installation, you can verify that the module, file context rules, and applied labels are present with the following commands:

```bash
# semodule -lfull | grep ganeti
400 ganeti               pp

# semanage fcontext -l | grep ganeti
/usr/lib/ganeti/[^/]+/usr/sbin/ganeti-kvmd         regular file       system_u:object_r:ganeti_kvmd_exec_t:s0
/usr/lib/ganeti/[^/]+/usr/sbin/ganeti-luxid        regular file       system_u:object_r:ganeti_luxid_exec_t:s0
/usr/lib/ganeti/[^/]+/usr/sbin/ganeti-wconfd       regular file       system_u:object_r:ganeti_wconfd_exec_t:s0

# ls -Z /usr/lib64/ganeti/*/usr/sbin/ganeti-* | grep ":ganeti_"
  system_u:object_r:ganeti_kvmd_exec_t:s0 /usr/lib64/ganeti/3.1/usr/sbin/ganeti-kvmd
 system_u:object_r:ganeti_luxid_exec_t:s0 /usr/lib64/ganeti/3.1/usr/sbin/ganeti-luxid
system_u:object_r:ganeti_wconfd_exec_t:s0 /usr/lib64/ganeti/3.1/usr/sbin/ganeti-wconfd
  system_u:object_r:ganeti_kvmd_exec_t:s0 /usr/lib64/ganeti/default/usr/sbin/ganeti-kvmd
 system_u:object_r:ganeti_luxid_exec_t:s0 /usr/lib64/ganeti/default/usr/sbin/ganeti-luxid
system_u:object_r:ganeti_wconfd_exec_t:s0 /usr/lib64/ganeti/default/usr/sbin/ganeti-wconfd
```
